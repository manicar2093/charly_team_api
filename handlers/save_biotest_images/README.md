# Save biotest images

## Request

```json
{
    "biotest_uuid": "biotest_uuid",
    "front_picture": "front/image/path",
    "back_picture": "back/image/path",
    "right_side_picture": "right/image/path",
    "left_side_picture": "left/image/path"
}
```

biotest_uuid is required

## Response

```json
{
    "code": 200,
    "status": "OK"
}
```
